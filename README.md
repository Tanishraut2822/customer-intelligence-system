# Customer Intelligence System

An end-to-end machine learning project to analyze customer behavior using RFM analysis, predict future purchases, and recommend products.

## Features
- Customer segmentation using K-Means clustering
- Future purchase prediction using Random Forest
- Product recommendation using collaborative filtering
- Interactive Power BI dashboard
- Streamlit web application for predictions

## Tech Stack
- Python, Pandas, NumPy
- Scikit-learn
- Power BI
- Streamlit
- VS Code

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
